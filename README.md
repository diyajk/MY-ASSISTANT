# 🚀 Max AI Productivity Assistant

### 🤖 Your Personal Productivity Partner

Max is an AI-powered productivity assistant built with **Python** and **Google Gemini 2.5 Flash** to help users stay organized, focused, and efficient throughout the day.

## ✨ Features

* 📅 Smart schedule planning
* ✅ Task breakdown and prioritization
* 🎯 Productivity and focus coaching
* 🧠 Context-aware conversations
* 💬 Session-based memory
* ⚡ Fast AI-powered responses
* 📋 Daily planning and workflow organization

## 🛠️ Tech Stack

* 🐍 Python
* 🤖 Gemini 2.5 Flash
* 📦 Google GenAI SDK
* 💻 Visual Studio Code

## 🚀 Getting Started

```bash
pip install google-genai
python assistant.py
```

## 💡 About the Project

Max is designed to be more than just a chatbot. It acts as a productivity assistant that helps organize tasks, create actionable plans, manage schedules, and keep users focused on their goals.

Whether you're planning your day, managing projects, or breaking down complex tasks, Max provides intelligent assistance powered by Gemini AI.
